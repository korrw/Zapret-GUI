# ZAPRET GUI

Графический интерфейс для [Zapret](https://github.com/flowseal/zapret-discord-youtube) - инструмента обхода блокировок Discord и YouTube.

## Особенности

- Современный минималистичный интерфейс в черно-белой теме
- Поддержка всех 19 конфигураций Zapret
- Простое управление одной кнопкой
- Автоматическое управление процессами
- Встроенный оригинальный Zapret

## Установка

1. Скачай установщик из [Releases](https://github.com/korrw/Zapret-GUI/releases)
2. Запусти `ZAPRET Setup 1.0.0.exe`
3. Следуй инструкциям установщика

## Использование

1. Запусти приложение ZAPRET
2. Выбери конфигурацию из списка
3. Нажми на кнопку питания для запуска
4. Для остановки нажми кнопку снова или закрой приложение

## Доступные конфигурации

- General (основная)
- General ALT (1-11)
- Fake TLS (AUTO, ALT, ALT2, ALT3)
- Simple Fake (основная, ALT, ALT2)
